# Predicting-Breast-Cancer
Used breast cancer dataset from Kaggle and performed classifiaction i.e. to predict whehter the cancer is benign or malignant.
## Main Feature:
- Performed Exploratory Data Analysis
- Performed feature selection using feature_selection library from sklearn
- Encoded Categorical Target variable
- Trained and Compared Predictive Models
    - Logistic Regression - 97.4% Accuracy
    - Decision Tree Classifier - 92.1% Accuracy
    - RandomForest Classifer - 95.6% Accuracy
    - Bagging Classifier - 96.5% Accuracy
